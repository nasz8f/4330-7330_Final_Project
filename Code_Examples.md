## Switch Statement

C++

    switch(x)
    {
      case 1:
      //Do Something
      case 2:
      //Do Something Else
    }
    
C#

    switch(x)
    {
      case1:
      case2:
      case3:
      //Do Something
    }
    
C# can only fall through to the next case if the case doesn't do any work.
